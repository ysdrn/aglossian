  2. 「行為と解明──アンスコムは「反因果説」ではない」, 『哲学の探求』, 哲学若手研究者フォーラム, 48号, pp.198-213, 2021.

    アンスコムは反因果説の立場を採るとしばしば言われるが、この読解を正当化するテクストは存在しない。それどころか彼女は『インテンション』において、反因果説の根幹的テーゼたる原因と理由の峻別を否定している。デイヴィドソンとアンスコムは、ある哲学的テーゼを支持するかどうか（因果説・反因果説）で対立しているのではなく、方法論において（分析・解明）対立している。アンスコムの特異な哲学的方法論として「解明」を取り出し、彼女の実践的知識論をこの観点から読解する。
